# fract-ol

Explore the wonderfull world of fractals.

### Building 

`make bonus`


### Execution

`./fract-ol <FRACTAL_TYPE>`

#### Available types
  - Julia
  - Mandelbrot
  - Tricorn


### Controls

`mouse scroll` : zoom in and out the fractal (endlessly)

`arrow keys` : move the fractal

`A` `S` : change color range
